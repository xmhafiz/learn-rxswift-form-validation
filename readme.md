## iOS Sample form validation using RxSwift
Single and multiple fields validation for learning purposes

### Specification

- Xcode 11+
- Swift 5+
- Cocoapods

### Common Pods
- [RxSwift](https://github.com/ReactiveX/RxSwift)

Run `pod install` and open the .workspace file
